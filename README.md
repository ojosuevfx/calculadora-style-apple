# Calculadora Apple Glass

Projeto desenvolvido por **Josué Ribeiro** - Uma calculadora moderna com design inspirado na Apple e efeito liquid glass.

## Como executar o projeto?

O projeto requer Node.js & npm instalados - [instale com nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga estes passos:

```sh
# Passo 1: Clone o repositório.
git clone <YOUR_GIT_URL>

# Passo 2: Navegue até o diretório do projeto.
cd <YOUR_PROJECT_NAME>

# Passo 3: Instale as dependências necessárias.
npm i

# Passo 4: Inicie o servidor de desenvolvimento com auto-reload e preview instantâneo.
npm run dev
```

## Quais tecnologias são usadas neste projeto?

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como fazer deploy deste projeto?

Você pode fazer deploy usando qualquer plataforma de hospedagem estática como:
- Vercel
- Netlify
- GitHub Pages
- Cloudflare Pages

Basta fazer o build do projeto com `npm run build` e fazer upload da pasta `dist` gerada.
